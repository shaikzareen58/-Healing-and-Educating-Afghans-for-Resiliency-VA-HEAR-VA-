# Healing and Educating Afghans for Resiliency -- VA HEAR VA
This project focuses on managing EHR workflows and maintaining structured patient data systems for refugee wellness care.
The objective is to ensure accurate documentation, effective patient tracking and organized follow-up processes.
Key tasks include maintaining REDCap databases, tracking enrollment and outcomes, generating reports and coordinating outreach data collection.
The project also supports workflow standardization and quality improvement to enhance clinic efficiency and data reliability.
